What is Booky?

Booky is a sophisticated conversational AI chatbot that specializes in all things books, bringing you the most comprehensive and intelligent book discovery experience.

Conversational AI

A specialized chatbot that understands natural language and engages in meaningful conversations about books, authors, and literature.

Comprehensive Queries

Answers questions about authors, genres, pricing, ratings, availability, and provides detailed book information from multiple sources.

Real-Time Data

Retrieves live metadata from Google Books API, ensuring you get the most current information about pricing, ratings, and availability.

Advanced AI

Uses Rasa for intent detection and LangChain RAG for contextual reasoning, providing accurate and relevant responses.

Modular Architecture

Built with a scalable, modular architecture using FastAPI, Redis caching, and multiple data sources for optimal performance.

Rich Dataset

Combines a structured dataset of 10,000+ books with real-time API data for the most comprehensive book information.
